# app-mgmt-api

## 概要
app-mgmt-web用のAPIです。  

## 使用技術
- TypeScript v4.5.5
- Node.js v18.0.0
- NestJS
- GraphQL
- MySQL（開発環境）
- PostgreSQL（本番環境）
- TypeORM v0.2.44  

## インフラ
- Heroku
- Docker Compose（開発環境）  

## 作成理由
